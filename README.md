# Компьютерная мини-игра на WinForms || A simple computer game on WinForms
В данном репозитории представлена простая компьютерная игра *Автогонки*, реализованная при на языке C# с применением создания оконных приложений от компании Microsoft - Windows Forms.
## Минимальные требования
.NET Framework >= 4.7.2
## Правила игры
Задача обоих игроков уворачиваться от летящих на них препятствий. Тот игрок, кто первым столкнётся с препятствием, - проиграл.
Также игрокам запрещается пересекаться друг с другом, таким образом, игроки должны соревноваться друг с другом и помогать одновременно.

This repository presents a simple computer game *Car Racing*, implemented in C# using the creation of windows applications from Microsoft - Windows Forms.
## Minimum requirements
.NET Framework version no less than 4.7.2
## Game rules
The task of both players is to dodge the obstacles flying at them. The player who is the first to encounter an obstacle has lost.
It is also forbidden for players to cross paths with each other, so players must compete with each other and help at the same time.
